# Playing with Github API Backend

> HTTP API to search for Github repositories.

## Installation

OS X & Linux:

```sh
cd backend
npm install
```

## Start Service

```sh
npm start
```

## Test the API

The API can be test with swagger in the url: https://localhost:3000/api-docs when the server is running.
